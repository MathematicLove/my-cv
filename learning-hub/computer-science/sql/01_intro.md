## Основы синтаксиса и базовые запросы

### Подключение к СУБД через CLI

```zsh
psql -U USERNAME
```

Или подключение к конкретной базе данных:
```zsh
psql -U USERNAME -d DB_NAME
```

Далее ввод пароля пользователя.
### Просмотр всех баз данных
```psql
\l
```
или 
```psql
\list
```

### Просмотр таблиц в конкретной базе данных

```psql
\dt
```
Более подробно:
```psql
\dt+
```
Для всех схем:
```psql
\dt *.*
```

Вывести список всех таблиц, представлений (views), последовательностей (sequences) и других отношений в текущей схеме, с указанием их типа и владельца:
```psql
\d
```
### Создание базы данных
```sql
CREATE DATABASE DB_NAME;
```

Так же можно создавать БД, из CLI: 
```zsh
createdb DB_NAME
```

Или с опциями:
```zsh
createdb -O OWNER_NAME DB_NAME
```

Пример создания:
```sql 
CREATE TABLE TABLE_NAME IF NOT EXISTS (
	user_id SERIAL PRIMARY KEY,
	username VARCHAR(255) NOT NULL,
	balance DECIMAL
);
```
### Удаление БД
```sql
DROP DATABASE DB_NAME;
```

Можно через CLI:
```zsh
dropdb -U DATABASE_NAME
```
### Удаление таблиц

```sql
DROP TABLE IF EXISTS TABLE_NAME;
```
